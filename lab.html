Game of life

Ce project est un exercice que nous avons réalisé avec notre prof pendant la période de confinement.

Reproduire le comportement de cette algorithme de John Horton Conway dans un navigateur à l'aide de three.js nous a permit de :

  • Travailler l'algorithmie
  • Mettre en application les fondamentaux de la 3d et three.js
  • Se familiariser avec Wepback et les differentes configuration de build
J'ai beaucoup aimé cette exercice, on a utilisé pleins de concepts de programmation et de js pour parvenir au résultat voulu, c'était très complet et agréable à faire.

Trombinoscope Vue js

Lors du confinement, nous avons eu un gros project à réaliser. Nous devions réaliser un trombinoscope des anciens élèves du départment.

Étant donné que la façon de réaliser ce trombinoscope n'était pas dans les consignes j'ai choisi d'en profiter pour commencer, seul, un framework js : Vue.js. J'ai donc pu aborder ces points :

  • Utilisation du Vue CLI 3 & Vue UI
  • Principe de cycle de vie des components
  • Utilisations des templates, des props, directives..
  • Fondamentaux Firebase.

Le project n'est pas encore fini, la page de ce project sera disponible dès que j'aurais l'autorisation de partager le projet.

Chat Node.js

Projet réalisé lors du confinement, l'utilisation de la doc, les tests...

Nous avons donc réalisé un chat node js très simple, sans historique en utilisant Express et socket.io

  • Fondamentaux Nodejs et async javascript
  • Set up d'un server en Node avec Express
  • Utilisation de socket io et des emits cheatsheet
  • Deploiement du server Node.js avec heroku

Pour le moment la console est executé coté client, mais avec les différents façon d'emit de socket io, il est assez simple d'executer la console coté server et rajouter quelques commandes (/mute...)
