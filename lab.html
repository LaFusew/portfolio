Game of life

Ce projet est un exercice que nous avons réalisé avec notre prof pendant la période de confinement.

Reproduire le comportement de cette algorithme de John Horton Conway dans un navigateur à l'aide de three.js nous a permis de :

  • Travailler l'algorithmie
  • Mettre en application les fondamentaux de la 3d et three.js
  • Se familiariser avec Wepback et les differentes configuration de build
J'ai beaucoup aimé cette exercice, on a utilisé pleins de concepts de programmation et de js pour parvenir au résultat voulu, c'était très complet et agréable à faire.

Trombinoscope Vue js

Lors du confinement, nous avons eu un gros projet à réaliser, nous devions réaliser un trombinoscope des anciens élèves du département.

Étant donné que la façon de réaliser ce trombinoscope n'était pas dans les consignes j'ai choisi de faire ma premier SPA en Vue.js pour ce projet. Cela m'a permis d'aborder seul ces points

  • Utilisation du Vue CLI 3 & Vue UI
  • Principe de cycle de vie des components
  • Utilisations des templates, des props, directives..
  • Fondamentaux Firebase.
Je n'ai par contre pas utilisé VueX.
Le project n'est pas encore fini, la page pour visualisé ce project sera disponible dès que je l'aurais fini et que j'aurais demandé l'autorisation à l'école de partager ce trombinoscope.

Chat Node.js

Exercice réalisé lors du confinement, notre professeur nous à fait faire beaucoup de choses différentes car il important de comprendre comment chercher a maitriser une techno, l'utilisation de la doc, les tests...

Nous avons donc réalisé un chat node js très simple, sans historique en utilisant Express et socket.io

  • Fondamentaux Nodejs et async javascript
  • Set up d'un server en Node avec Express
  • Utilisation de socket io et des emits cheatsheet
  • Deploiement du server Node.js avec heroku
Pour le moment la console est executé coté client, mais avec les différentes façons d'emit de socket io, je compte passer l'execution de la console coté server et rajouter qql commandes (/mute...) Voir la page du projet
